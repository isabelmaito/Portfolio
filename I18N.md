# Sistema de Internacionalização (i18n)

## 📋 Visão Geral

Este projeto implementa um sistema de internacionalização (i18n) que permite alternar facilmente entre **português (pt)** e **inglês (en)** em todo o aplicativo.

## 🏗️ Estrutura

```
src/
├── context/
│   ├── LanguageContext.jsx    # Contexto de linguagem
│   └── ThemeContext.jsx       # Contexto de tema (existente)
├── locales/
│   └── translations.js        # Todas as traduções
├── components/
│   └── Header.jsx             # Botão de toggle de idioma
└── pages/
    ├── Home.jsx
    ├── Sobre.jsx
    ├── Habilidades.jsx
    └── Contato.jsx
```

## 🚀 Como Usar

### Em um Componente

```jsx
import { useLanguage } from '../context/LanguageContext';

function MyComponent() {
  const { t, language, toggleLanguage } = useLanguage();

  return (
    <div>
      <h1>{t('home.greeting')}</h1>
      <p>Idioma atual: {language === 'pt' ? 'Português' : 'English'}</p>
      <button onClick={toggleLanguage}>Trocar Idioma</button>
    </div>
  );
}
```

### Estrutura de Chaves de Tradução

As traduções são organizadas hierarquicamente:

```javascript
{
  pt: {
    header: {
      home: 'Início',
      about: 'Quem sou eu',
      // ...
    },
    home: {
      greeting: 'Olá, eu sou Isabel Maito',
      // ...
    },
    // ...
  },
  en: {
    // ... mesma estrutura em inglês
  }
}
```

## 📝 Adicionando Novas Traduções

1. **Abra** [src/locales/translations.js](src/locales/translations.js)
2. **Encontre** a seção apropriada (ex: `home`, `about`, `skills`, `contact`)
3. **Adicione** a chave em português e inglês:

```javascript
// Exemplo
export const translations = {
  pt: {
    home: {
      newField: 'Novo Campo',
      // ...
    }
  },
  en: {
    home: {
      newField: 'New Field',
      // ...
    }
  }
}
```

4. **Use** no seu componente:

```jsx
<div>{t('home.newField')}</div>
```

## 🔄 Alternância de Idioma

- **Botão** na barra de navegação (header) mostra `PT` ou `EN`
- **Clique** para alternar entre português e inglês
- **Preferência** é salva no `localStorage`
- **Página inteira** é atualizada em tempo real (zero delay)

## 💾 Persistência

A linguagem selecionada é automaticamente salva em `localStorage` com a chave `language`. Quando o usuário retorna ao site, o idioma preferido é carregado.
