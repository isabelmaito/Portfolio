# Sistema de Internacionalização (i18n) com i18next

## 📋 Visão Geral

Este projeto implementa um sistema de internacionalização robusto usando **i18next** e **react-i18next**, permitindo alternar facilmente entre **português (pt)** e **inglês (en)** em todo o aplicativo.

## 🏗️ Estrutura

```
src/
├── i18n.js                    # Configuração do i18next
├── context/
│   ├── LanguageContext.jsx    # Wrapper para facilitar uso do i18next
│   └── ThemeContext.jsx       # Contexto de tema (existente)
├── locales/
│   ├── pt.json                # Traduções em português
│   └── en.json                # Traduções em inglês
├── components/
│   └── Header.jsx             # Botão de toggle de idioma
└── pages/
    ├── Home.jsx
    ├── Sobre.jsx
    ├── Habilidades.jsx
    └── Contato.jsx
```

## 🚀 Como Usar

### Em um Componente

```jsx
import { useLanguage } from '../context/LanguageContext';

function MyComponent() {
  const { t, language, toggleLanguage } = useLanguage();

  return (
    <div>
      <h1>{t('home.greeting')}</h1>
      <p>Idioma atual: {language === 'pt' ? 'Português' : 'English'}</p>
      <button onClick={toggleLanguage}>Trocar Idioma</button>
    </div>
  );
}
```

### Uso Direto do i18next (Opcional)

```jsx
import { useTranslation } from 'react-i18next';

function MyComponent() {
  const { t, i18n } = useTranslation();

  return (
    <div>
      <h1>{t('home.greeting')}</h1>
      <button onClick={() => i18n.changeLanguage('en')}>English</button>
    </div>
  );
}
```

## 📝 Adicionando Novas Traduções

1. **Abra** ambos os arquivos [src/locales/pt.json](src/locales/pt.json) e [src/locales/en.json](src/locales/en.json)
2. **Encontre** a seção apropriada (ex: `home`, `about`, `skills`, `contact`)
3. **Adicione** a chave em ambos os idiomas:

**pt.json**
```json
{
  "home": {
    "newField": "Novo Campo"
  }
}
```

**en.json**
```json
{
  "home": {
    "newField": "New Field"
  }
}
```

4. **Use** no seu componente:

```jsx
<div>{t('home.newField')}</div>
```

## 🔄 Alternância de Idioma

- **Botão** na barra de navegação (header) mostra `🇧🇷 PT` ou `🇬🇧 EN`
- **Clique** para alternar entre português e inglês
- **Preferência** é salva no `localStorage`
- **Página inteira** é atualizada em tempo real (zero delay)

## 💾 Persistência

A linguagem selecionada é automaticamente salva em `localStorage` com a chave `language`. Quando o usuário retorna ao site, o idioma preferido é carregado automaticamente pelo i18next.

## ⚙️ Configuração

O i18next está configurado em [src/i18n.js](src/i18n.js):

```javascript
import i18n from 'i18next';
import { initReactI18next } from 'react-i18next';
import ptTranslations from './locales/pt.json';
import enTranslations from './locales/en.json';

i18n
  .use(initReactI18next)
  .init({
    resources: {
      pt: { translation: ptTranslations },
      en: { translation: enTranslations }
    },
    lng: localStorage.getItem('language') || 'pt',
    fallbackLng: 'pt'
  });
```

## 🔧 Funcionalidades Avançadas do i18next

O i18next já oferece suporte nativo para:

1. **Interpolação**: Inserir variáveis dinâmicas
   ```jsx
   // en.json: "welcome": "Welcome, {{name}}!"
   {t('welcome', { name: 'Isabel' })}
   ```

2. **Pluralização**: Textos que mudam no singular/plural
   ```json
   {
     "items": "{{count}} item",
     "items_plural": "{{count}} items"
   }
   ```

3. **Formatação**: Datas, números e moedas
   ```jsx
   {t('date', { date: new Date() })}
   ```

4. **Namespaces**: Separar traduções por módulos/features

5. **Detecção automática**: i18next-browser-languagedetector para detectar idioma do navegador

## 🌍 Adicionando Mais Idiomas

Para adicionar um novo idioma (ex: espanhol):

1. Crie [src/locales/es.json](src/locales/es.json)
2. Adicione ao [src/i18n.js](src/i18n.js):
   ```javascript
   import esTranslations from './locales/es.json';
   
   resources: {
     pt: { translation: ptTranslations },
     en: { translation: enTranslations },
     es: { translation: esTranslations }
   }
   ```
3. Atualize o botão de idioma no Header para suportar 3 opções
